Tipos de Caixas

Caixas do tipo box-level

Elementos box-level, também conhecidos como elementos de bloco, ocupam toda a largura disponível do seu elemento pai (container), criando uma quebra de linha antes e depois do elemento, como se fossem blocos empilhados verticalmente.

<div>
<h1> até <h6>
<p>
<main>
<header>
<nav>
<article>
<aside>
<footer>
<form>
<video>

Caixas do tipo inline-level

Elementos inline-level, também conhecidos como elementos em linha, ocupam apenas o espaço necessário para o seu conteúdo, fluindo horizontalmente com o texto adjacente, como palavras em uma frase.

<span>
<a>
<code>
<small>
<strong>
<em>
<sup> - <sub>
<label>
<button>
<input>
<select>
