Listas ordenadas

Temos a possibilidade de utilizar a tag <ol> para criar uma lista ordenada.

Dentro da tag temos o parâmetro type="" que apontamos como queremos a indicação dos itens.


  1. Podemos apontar como "I" para indicar números romanos maiúsculos;
  1. Podemos apontar como "i" para indicar números romanos minúsculos;
  1. Podemos apontar como "A" para indicar letras maiúsculas;
  1. Podemos apontar como "a" para indicar letras minúsculas.

Porém se indicarmos o parâmetro type="", a nossa lista será enumerada.

  1. Acordar
  2. Ligar para o João
  3. Tomar café
  4. Escovar os dentes
  5. Ir para a faculdade
  6. Almoçar
  7. Ir para o trabalho
  8. Voltar para casa
  9. Jantar
  10. Dormir

Além do parâmetro type="", temos o parâmetro start="" que indica em que posição nossa lista irá começar.

  1. Acordar

Observação importante: No HTML5 não se faz necessário o fechamento da tag <li> (</li>), por isso tanto faz criar a lista com o fechamento da tag ou não. Porém, por questão de organização do código, usar o mesmo padrão para todo o código. Por exemplo, se eu vou escrever o código fechando a tag <li>, todas as tag <li> deverão ser fechadas (</li>), caso o contrário, todas as tag <li> deverão ser abertas.

Listas não ordenadas

Temos a possibilidade de utilizar a tag <ul> para criar uma lista não ordenada.

Dentro da tag temos o parâmetro type="" igual a tag <li>, porém nós utilizamos as opões "disc", "circle" e "square".

Se não alimentarmos o type="" com nenhuma opção, a opção "disc" será adotada como padrão.

Exemplo 1: <ul>
  • Pão
  • Leite
  • Tomate
  • Manteiga
  • Arroz
  • Feijão

Exemplo 2: <ul type="disc">
  • Pão
  • Leite
  • Tomate
  • Manteiga
  • Arroz
  • Feijão

Exemplo 3: <ul type="circle">
  • Pão
  • Leite
  • Tomate
  • Manteiga
  • Arroz
  • Feijão

Exemplo 4: <ul type="square">
  • Pão
  • Leite
  • Tomate
  • Manteiga
  • Arroz
  • Feijão

Lista de definição

Temos a possibilidade de utilizar a tag <dl> para criar uma lista de definição.

Dentro da tag <dl>, usamos a tag <dt> para o termo e <dd> para a descrição

Exemplo:

HTML
Linguagem de marcação para a criação do conteúdo de um site.
CSS
Linguagem de marcação para a criação do design de um site.
JavaScript
Linguagem de programação para a criação de interatividade de um site.

<dl>
    <dt>HTML</dt>
    <dd>Linguagem de marcação para a criação do conteúdo de um site.</dd>
    <dt>CSS</dt>
    <dd>Linguagem de marcação para a criação do design de um site.</dd>
    <dt>JavaScript</dt>
    <dd>Linguagem de programação para a criação de interatividade de um site.</dd>
</dl>
